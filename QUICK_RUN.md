# Quick Run Guide

## Choose One Backend:

### Node.js Backend (Port 5000)
```bash
cd backend
npm install
npm run dev
```

### Django Backend (Port 8000)
```bash
cd backend\backend
..\venv\Scripts\activate
python manage.py runserver
```

## Run Frontend (Port 3000)
```bash
npm install
npm start
```
